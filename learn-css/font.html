he survey also examined the usage of various CSS features. While filter effects were among the most popular, some underutilised features drew attention. Hanging punctuation, for example, had a low usage rate of 1.57%, but more than half of the respondents expressed an interest in learning more about it. Similarly, anchor positioning had a 4.8% usage rate, although this figure may rise as browser support grows and more educational content becomes available. Cascade Layers, which simplify CSS management, were used by 18.7% of respondents. This feature, along with @scope (also at 4.8% usage), suggests that newer tools aimed at improving CSS organisation are slowly gaining traction.
